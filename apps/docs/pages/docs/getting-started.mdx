# Getting Started

Wizard UI depends on Chakra UI, React Query and @cosmjs.

## Installation

Install Wizard UI and make sure you install its peer dependencies. It should be done automaticaly with npm v7.

```plaintext
npm install @wizard-ui/core @wizard-ui/react
```

## Quick Start

First, add the `ConnectionProvider`, `WalletProvider` and `WalletModalProvider` to your app.

```tsx {4,5,17,19}
import { ChakraProvider } from "@chakra-ui/react";
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";
import {
  ConnectionProvider,
  WalletProvider,
  WalletModalProvider,
  theme,
} from "@wizard-ui/react";

const queryClient = new QueryClient();

function App() {
  return (
    <ChakraProvider theme={theme}>
      <ConnectionProvider endpoint={endpoint}>

          <WalletModalProvider>
            <Component />

        </WalletProvider>
      </ConnectionProvider>
    </ChakraProvider>
  );
}
```

Finally, use hooks! Let's check balance of the current wallet.

```tsx {4,5,17,19}
import { useBalance, useAddress } from "@wizard-ui/react";

function Profile() {
  const { address } = useWallet();
  const { data: ulunaBalance } = useBalance({ address, token: "uosmo" });

  if (address == null) {
    return <div>Not connected</div>;
  }

  return (
    <div>
      Connected to {address} with {ulunaBalance} LUNA
    </div>
  );
}
```
